vkfeed2rss - программа на php, переводящая стены страниц ВКонтакте в RSS ленты.

Требуется php 7.0+.

Программа находится в альфа-стадии и постепенно улучшается.

Что может:
* Ест любые ссылки: vk.com/id1, id1, http://vk.com/id1
* Парсит так и обычные посты, так и репосты. Потихоньку программа делится на функции.
* Картинки в виде HTML вставок
* Гифки как картинки
* Видео, в виде ссылок с превьюшкой
* Аудио, но не работает, т.к. вк зОпретил
* Документы (кроме гифок) как ссылки
Что не может:
* Не придумывает заголовки, а просто даёт ID записи
* Не работает без API ключа (никак не исправить)

Использование:
vkfeed2rss.php[?опции]

&url - ссылка на страницу, ест полные и неполные ссылки:
	"http://vk.com/id1",
	"vk.com/id1",
	"id1"
	Не переваривает "//vk.com/id1" и "/id1". Давать предпоследний рекомендуется опцией ниже. &url предпочитайте не использовать, т.к. кривой.
&path - как и &url, но переваривает только доменные ссылки: "id1", "apiclub", "chechnya", "club1". Стабилен.
&count - количество записей. Не больше 100, не меньше 0. Если 0, то значение по-умолчанию.
&filter - фильтр. Документация в vkapidoc://wall.get.

Если удумаете заниматься разработкой, то напомню, что ВСЕ api запросы должны быть с опцией "v" и иногда с "access_token". Иначе будут проблемы.

Добавлю то, что для некоторых функций нужен ключ API, поэтому его нужно скормить программе. В "conf/vkapikey" засуньте ключ.
